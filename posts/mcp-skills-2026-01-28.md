## What Are MCP Skills?

MCP (Model Context Protocol) skills are modular tools that extend AI assistants:

- **Web scraping** (Firecrawl, Puppeteer)
- **File system access** (filesystem-mcp)
- **Database queries** (postgres-mcp)
- **GitHub integration** (github-mcp)
- **Browser automation** (puppeteer-mcp)
