## What Are Clawdbot Skills?

Clawdbot skills are local automation capabilities that extend Clawdbot's functionality. Each skill:
- Has a `SKILL.md` file defining its purpose and usage
- Lives in `/home/clawdbot/clawd/skills/` (local) or `/usr/lib/node_modules/clawdbot/skills/` (system)
- Can be invoked by the clawdbot agent for specific tasks

## Adding New Skills

1. Create a skill directory: `/home/clawdbot/clawd/skills/<skill-name>/`
2. Add `SKILL.md` with:
   - Description
   - Installation commands
   - Usage examples
3. The skill is automatically discovered
